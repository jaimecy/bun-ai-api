# bun-ai-api

API de IA

## Instalación

Para instalar las dependencias:

```bash
bun install
```

## Ejecución

Para ejecutar el servidor:

```bash
bun run index.ts
```

O en modo desarrollo:

```bash
bun run dev
```

Este proyecto fue creado usando `bun init` en bun v1.3.5. [Bun](https://bun.com) es un runtime JavaScript rápido todo-en-uno.
